This documentation aims at helping you integrate our MyScript widgets and components into your applications. It is intended for iOS developers with good knowledge of object-oriented programming. To install and integrate our MyScript widgets and components on iOS, you must be using the latest version of Xcode and iOS SDK. Minimum deployment target is 7.0.

To understand how our widgets interact with each other and how that may impact your work, see the Dependencies section.

Installation

  1. Unzip the archive folder you have downloaded on the developer portal (MyScript_ATK-ios.zip) to obtain the following tree:
    • MyScript_ATK-ios-1.3.0
      • doc
        Contains the whole documentation (installation, functional, reference).
      • libs
        Contains the libraries and binaries.
      • samples
        Contains the samples, showing basic use cases.
      • third_part
        Contains the ATK and third-party licenses.
      • atk-info.json
        Contains information about the version numbers.
  2. Download your certificate.
    To learn how to create and download a certificate, follow this procedure.
