MyScript® Application Toolkit (ATK)

Deployment

Internet access

An Internet connection is required to launch and run ATK applications.
However, a 30-day grace period is offered, from the moment you first launch your application. A connection will then be mandatory.

Certificate

A certificate is required to use the MyScript handwriting recognition engine. It gives the right to use ATK. Each certificate gives access to all widgets for a given platform.

You need one certificate per application and per platform. You can integrate several widgets in one application.

To download the samples' certificate, follow this procedure:

  1. Go to https://atk.myscript.com.
  2. Enter your email and password.
  3. Click Open on the ATK Samples application.
  4. Click on the download icon to download the MyCertificate.h file.
  5. Paste it in the samples folder and thus replace the existing MyCertificate.h file.
  6. Open the corresponding *.xcodeproj file in the samples folder.
  7. Build and run the sample app.

To download any other certificate, follow this procedure:

  1. Go to https://atk.myscript.com.
  2. Enter your email and password.
  3. In the Applications tab, click Create application.
  4. Enter your application name (at least 5 characters) and its description, then click Create.
  5. Click Open, then Create certificate.
  6. Enter the bundle identifier that can be found in your project's properties (General tab) and select the target environment, then click Create.
  7. Click on the download icon to download the corresponding *.h certificate.
  8. Paste it in your project and start creating your app.

Runtime management

A runtime is the first launch of one application on one device.

You are allowed 40 runtimes by default, which enable you to install and run one application on 40 devices.

Go to the Applications tab of the Admin User Interface to see your remaining runtimes.

Go to the developer portal to buy additional cartridges (either 500 or 1,000 units).

If you install and run the samples, your runtime account will be deducted.
