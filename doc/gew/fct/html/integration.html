Geometry Widget

To install and integrate the Geometry Widget on iOS, you must be using the latest version of Xcode and iOS SDK. Minimum deployment target is 7.0.

    Integration

    Links between projects

    The Geometry Widget for iOS is a framework that you add to your Xcode project.

    • Click on your project in the project navigator.
    • Choose the relevant target.
    • Open the Build Phases tab.

    • In the Link Binary With Libraries section, click on the + sign and add SystemConfiguration.framework.
    • Click again on the + sign add the following MyScript framework and libraries:
      • AtkGew.framework
      • libAtkInw.a
      • libMyScript2D.a
      • libMyScriptDocument.a
      • libMyScriptEngine.a
      • libMyScriptInk.a
      • libMyScriptShape.a
      • libMyScriptText.a
      • libStylusCore.a

    • In the Build Settings tab of your target, look for Other Linker Flags and add the following flags:
      • -ObjC
      • /usr/lib/libstdc++.dylib
      • -Wl,-U,_MyScript2D
      • -Wl,-U,_MyScriptAnalyzer
      • -Wl,-U,_MyScriptDocument
      • -Wl,-U,_MyScriptGesture
      • -Wl,-U,_MyScriptInk
      • -Wl,-U,_MyScriptInkSearch
      • -Wl,-U,_MyScriptMath
      • -Wl,-U,_MyScriptMusic
      • -Wl,-U,_MyScriptPrediction
      • -Wl,-U,_MyScriptShape
      • -Wl,-U,_MyScriptText
      • -Wl,-U,_MyScriptWriterAdaptation

    Bundle Resources

    The Geometry Widget requires some specific resources like images and fonts. To link these resources, add GeometryWidget.bundle to your project.

    Code sample

    To use the Geometry Widget in your app, add the following import instructions:

    #import <AtkGew/GeometryWidget.h>
#import "MyCertificate.h"

    You can instantiate a GWGeometryView with a few lines:

    // Create the GeometryWidget View
GWGeometryView *geometryView = [[GWGeometryView alloc] init];

// Recognition resources
NSArray *resources = @[@"shk-standard.res"];

// Certificate
NSData *certificate = [NSData dataWithBytes:myCertificate.bytes length:myCertificate.length];

// Configure the GeometryWidget View with the recognition resources and the certificate
[geometryView configureWithResources:resources
                                certificate:certificate];

    You can then use the GWGeometryView as any UIView.
