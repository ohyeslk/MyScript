Single Character Widget

Features

The Single Character Widget offers the following features:

General

    • Smooth ink capture and fade-out
    • Superimposed recognition mode (39 languages supported)
    • Isolated recognition mode for char-by-char mode and Letra only languages
    • Word and character candidates
    • Word completion
    • Fast and responsive, even with large amount of text
    • Multiple instances support (several widgets can run at the same time)
    • Gesture detection (single tap, long press, return, backspace, space)

API options

    • Character-based API
    • Non-blocking event-based API (50ms max. blocking time)
    • Simple configuration methods
    • Insert index management
    • Text editing
    • Word and character candidates
    • Word completion

Customization options

    • Ink color
    • Ink thickness
    • Ink fade-out delay
    • Ink effect
    • Background

    Languages

    The Single Character Widget supports:

      • a total of 39 languages in superimposed mode.
      • a total of 59 languages in isolated mode.

    If a language is not supported by the superimposed mode, the isolated mode will be activated by default.
