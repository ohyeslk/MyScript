Multi Line Text Widget

The Multi Line Text Widget (MLTW) for iOS is an interactive input method enabling users to write 1-page notes using handwriting. It is designed to ensure a natural, handwriting-based HMI with real-time recognition and ink editing gestures, thanks to a user-friendly interface.

The widgetÂ manages all the background processing attached to handwriting text on a single page. It lets you configure its behaviour and respond to the different available notifications issued from the recognition result, including editing gestures. Ink is recognized, saved as handwriting and can only be edited as such. Typeset notes are displayed only in the overview.

Here is a list of supported features:

    • Real-time handwriting recognition
    • Gesture detection
    • Editing gestures (erase, join, insert, overwrite, underline, selection, single tap, return)
    • Export as text or image
    • Faster and more efficient way to integrate input method
    • Powerful yet simple text editing API
    • Handwriting recognition setup with a single API call
    • Lots of UI customization options
