Multi Line Text Widget

Features

Modes

The Multi Line Text Widget can operate in 3 modes:

    • Writing mode: This is the default mode, it enables the user to write in a continuous flow.
    • Insertion mode: This mode enables the user to insert an extra space between two words.
    • Correction mode: This mode enables the user to insert a space within a word and thus edit the text already entered. It is also triggered when an item is selected.

Gesture detection

The below handwriting gestures are supported by the widget. A set of delegates is provided so that the input method can take action when a gesture is detected. The gestures' behaviour can be configured according to your needs, except for erase and overwrite gestures. You can also choose to activate or deactivate the gesture detection.

UI customization

A number of options are provided to customize the UI of the widget's writing area. The widget can be customized via the widget API.

The following items can be configured:

    • Ink width & color,
    • Background image of the input view,
    • Guidelines position, color & thickness,
    • Scrollbar background & drawable.

    Languages

    The Multi Line Text Widget supports 59 languages.
