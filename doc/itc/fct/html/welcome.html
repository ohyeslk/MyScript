Interactive Text Component

The Interactive Text Component (ITC) for iOS is an interactive input method enabling users to write notes using handwriting. It is designed to ensure a natural, handwriting-based HMI with real-time recognition and ink editing gestures, thanks to a user-friendly interface.

The component manages all the background processing attached to handwriting text on a page. It lets you configure its behaviour and respond to the different available notifications issued from the recognition result, including editing gestures. API parameters allow to control the different processing modes such as handwriting recognition, or real-time transformation of digital ink into typeset characters.

With its powerful background recognition capabilities, the component can associate recognized ink to smart user interface actions. Phone numbers, contact names, dates, etc. can be identified and highlighted as user feedback. Contextual actions can also be suggested on selected portions of digital ink, as launching a search request for instance.

The component provides no display capabilities, but it is really easy for developers to build one on top of it. It is actually designed for that purpose and the included code sample demonstrates this use case. Screenshots are taken from test applications and are only examples of what can be achieved with the component.

Here is a list of supported features:

    • Real-time handwriting recognition
    • Editing gestures (erase, join, insert, overwrite, underline, selection, single tap)
    • Notifications for ink, word and page modifications
    • Handwriting recognition setup with a single API call
    • Application sample with smart UI behaviour through data format identification and ink selection
