ITCPageInterpreter Class Reference

Inherits from NSObject
Declared in ITCPageInterpreter.h

Overview

The ITCPageInterpreter is the object in charge of configuration and the recognition of a page. It can deal with only one page at a time.

Do not inherit.

configurePageInterpreter:resources:lexicon:certificate:density:

Configures the handwriting recognition engine.

- (ITCError *)configurePageInterpreter:(NSString *)aLocale resources:(NSArray *)resources lexicon:(NSArray *)lexicon certificate:(NSData *)aCertificate density:(float)dpi

Parameters

resources

Array of paths to handwriting resource files.

lexicon

Array of user lexicon entries. May be nil.

dpi

The coordinate resolution in dot per inch.

locale

String representation of the handwriting recognition locale.

certificate

Data containing the handwriting recognition certificate.

Return Value

an ITCError

Discussion

This function is non-blocking and returns immediately. Configuration is a lengthy process that may take up to several seconds, depending on the handwriting resources to be configured. It is recommended to setup a configure listener to detect the end of the configuration process.

configurePageInterpreter:resources:lexicon:certificate:density:freezeTimeout:

Configures the handwriting recognition engine.

This function is non-blocking and returns immediately. Configuration is a lengthy process that may take up to several seconds, depending on the handwriting resources to be configured. It is recommended to setup a configure listener to detect the end of the configuration process.

- (ITCError *)configurePageInterpreter:(NSString *)aLocale resources:(NSArray *)resources lexicon:(NSArray *)lexicon certificate:(NSData *)aCertificate density:(float)dpi freezeTimeout:(NSInteger)freezeTimeout

Parameters

resources

Array of paths to handwriting resource files.

lexicon

Array of user lexicon entries. May be null.

dpi

The coordinate resolution in dot per inch.

freezeTimeout

The minimum freeze time before the launch of the recognition. It corresponds of the time between the last request and the launch of the recognition. Can be from 200 to 10000 ms. The default value is 650 ms.

locale

String representation of the handwriting recognition locale.

certificate

Byte array containing the handwriting recognition certificate.

certificateByteCount

Size of the byte array

Return Value

an ITCError

setGesture:defaultProcessing:

Sets the processing for the current gesture. The default processing is the default mecanism used to act according to the gesture. Else, it will be done in the implementation of gestures notifications (see SmartPage->setGestureDelegate)

- (void)setGesture:(ITCGestureType)gestureType defaultProcessing:(BOOL)defaultProcessing

Parameters

The current gesture (see ITCGestureType enum)

defaultProcessing

YES if you want the page interpreter to handle default processing of the detected gesture, NO if you want the page interpreter to delegate gesture processing to the application.
