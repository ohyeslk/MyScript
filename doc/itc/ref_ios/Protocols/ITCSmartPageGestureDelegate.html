ITCSmartPageGestureDelegate Protocol Reference

Conforms to NSObject
Declared in ITCSmartPageGestureDelegate.h

Overview

The ITCSmartPageGestureDelegate protocol defines the methods you can implement to be notified of the activity in a ITCSmartPage. These methods allow to monitor events related to the gestures.

All the methods in this protocol are optional.
