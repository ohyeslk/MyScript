Online accounts

Your email and password are the same for the Developer Portal and for the Admin User Interface.
Note that your profile can only be modified on the Developer Portal.

Developer Portal

Your Developer Portal account allows you:

    • to download development material and corresponding documentation,
    • to access the forum and support pages,
    • to purchase additional packages (for ATK) or cartridges (for CDK).

Admin User Interface

Your Admin User Interface account allows you:

    • to create, manage and obtain information on your applications and certificates,
    • to see your purchase history and consumption of runtimes.

To see a brief overview of the functionalities, go to the Admin User Interface section.
