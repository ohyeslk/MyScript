Single Line Text Widget

Features

Text entry

The Single Line Text Widget is designed to enable the user to write a single line of text in a continuous flow. The writing area scrolls automatically after a short delay once the user has finished writing. Scrolling of the writing area is managed by the widget.

Gesture detection

Handwriting gestures are detected by the widget. A set of delegate callbacks is provided so that the application can take action when some of the gestures are detected.

The following gestures are supported:

    • Single tap
    • Insert (top to bottom straight line)
    • Join (line from one word to the next one)
    • Return (continuous line from top to bottom then from right to left)
    • Erase (scratch-out or strike-through)
    • Overwrite

Text interface

The widget simplifies text manipulation by providing a simple character-based method to edit the text. Whether text is represented as ink or as characters in the writing area, the widget handles all of this transparently.

A single method is provided to modify the text of the writing area:

-(void)replaceCharactersInRange:(NSRange)range replacementString:(NSString *)text;

This method is a versatile tool that handles text insertion, deletion and replacement.
Handwriting recognition candidate change is implemented as a simple text replacement.

Handwriting recognition setup

The widget simplifies the setup of the handwriting recognition engine, and reconfiguration of the engine in case of a language change for example.

-(void)configureWithLocale:(NSString *)locale resources:(NSDictionary *)
resources lexicon:(NSArray *)lexicon certificate:(NSData *)certificate;

This function is non-blocking and performs configuration in the background to avoid blocking the UI thread. Delegate callbacks are provided to monitor the beginning and end of the configuration process and update the UI of the application accordingly.

UI customization

A number of options are provided to customize the UI of the widget's writing area.

The following items can be configured:

    • Font for typeset text
    • Text size
    • Text color
    • Ink width
    • Ink color
    • Background image of the scrolling area
    • Baseline position
    • Baseline thickness
    • Baseline color
    • Auto-scroll margin and delay (auto-scroll feature can be enabled/disabled)
    • Auto-typeset delay (auto-typeset feature can be enabled/disabled)

Languages

The Single Line Text Widget supports 59 languages.
