MAWSymbol Class Reference

Inherits from NSObject
Declared in MAWSymbol.h

Overview

A MAWSymbol represents a mathematical symbol that can be used for importing and exporting equations.

See [MAWMathViewController resultAsSymbolList], [MAWMathViewController addSymbol:allowUndo:] and [MAWMathViewController addSymbols:allowUndo:].
